Sample code for a harmonic oscillator for the V Encuentro Argentino de Mecánica geométrica y Física Matemática.

<https://sites.google.com/view/veamgfm/inicio>

It shows how to gradually improve a basic simulation.

### How to run this code

Run `uv sync` right after cloning the repo on your computer. Then try running the following commands.

```
uv run main.py
uv run parallel_example.py
uv run pytest
```
